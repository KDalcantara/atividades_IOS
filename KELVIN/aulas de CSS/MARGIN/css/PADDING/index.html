Pedding

CSS tem propriedades para especificar o preenchimento de cada lado de um elemento

Esse elemento div possui padding-top: 50 px; padding-right 30; padding-bottom: 50px; padding-left: 80px.

Preenchimento - propriedade abreviada

Para encurtar o codigo, é possivel especificar todas as propriedades de preenchimento em uma propriedade
